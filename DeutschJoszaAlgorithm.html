def sim_and_count(circuito, backend_usado='qasm_simulator'):
    """Correr um circuito num backend ou simulador e contar os resultados obtidos"""
    backend = Aer.get_backend(backend_usado)  # para correr num computador quntico, basta trocar para um backend a srio
    results = execute(circuito, backend=backend, shots=100).result()
    counts = results.get_counts()
    return counts

Porque adicionar C-NOT gates, no caso equilibrado? O nosso objetivo aplicar $U_f |x \rangle = (-1)^{f(x)}|x \rangle$; ou seja, a metade deles, queremos rodar o qubit. Com o C-NOT gate, isso que estaremos a fazer a metade deles, tendo a ancilla como controlo. Agora, se tu executares o cdigo e olhares para os C-NOT gates, vais ver algo que parece errado. Eles no deviam estar ao contrrio? No.

Passa-se que, quando temos um C-NOT, sendo o qubit q0 o controlo e o qubit q1 o que queremos atuar, se eles estiverem no meio de hadamard gates, o mesmo que ter apenas um C-NOT gate onde q1 o controlo e q0 onde estamos a atuar. Prova exemplo, com nomenclatura matemtica aldrabada (usando o operador $\hat{C}$ para simbolizar o C-NOT gate):

$\hat{H} \hat{C} \hat{H} |01\rangle = \hat{H} \hat{C} \hat{H} \begin{bmatrix}1\\0\end{bmatrix}\begin{bmatrix}0\\1\end{bmatrix} = \displaystyle{\frac{1}{\sqrt{2}}} \hat{H} \hat{C} \begin{bmatrix}1\\1\end{bmatrix}\begin{bmatrix}1\\-1\end{bmatrix} = \displaystyle{\frac{1}{\sqrt{2}}} \hat{H} \hat{C} \begin{bmatrix}1\\-1\\1\\-1\end{bmatrix} = \displaystyle{\frac{1}{\sqrt{2}}} \hat{H} \begin{bmatrix}1\\-1\\-1\\1\end{bmatrix} = \displaystyle{\frac{1}{\sqrt{2}}} \hat{H} \begin{bmatrix}1\\-1\end{bmatrix}\begin{bmatrix}1\\-1\end{bmatrix} = \displaystyle{\frac{1}{2}} \begin{bmatrix}0\\2\end{bmatrix}\begin{bmatrix}0\\2\end{bmatrix} = |11\rangle.$

Mas agora a verdadeira pergunta: "Porque raio que, ao encher aquilo de C-NOT gates, corresponde a equilibrado? como o termo ancillar 1, no estar a transformar tudo em 1?". Sim, transformar tudo em 1. No entanto, ests a olhar para isto do ponto de vista TOTAL! No TOTAL, sim, transformar tudo em 1. Mas s dentro do orculo, o que est l a acontecer? Quando os nossos qubits so 0, a ancilla no muda e, quando so 1, a ancilla muda. Tal como vimos na teoria, queremos alterar o termo ancillar e depois, devido a produtos tensoriais que esto aqui a acontecer, magia ir existir acima.
